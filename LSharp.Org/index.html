Abstract

L Sharp .NET is a powerful Lisp-like scripting language for .NET. It uses a Lisp dialect similar to Arc but tightly integrates with the .NET Framework which provides a rich set of libraries.

L Sharp is free software distributed under the terms of the GNU General Public License. You can download the latest release from Sourceforge. You can get the latest (potentially unstable!) source code by pointing your favourite Subversion client at https://svn.sourceforge.net/svnroot/lsharp. You may also want to read Rob Blackwell's Web Log.
