Acknowledgements

My thanks to all at Active Web Solutions for help, support, constructive comments and criticism.

Were it not for Paul Graham's book, Hackers and Painters, L Sharp would probably never have been born.

The following books have been particularly helpful:

Applied Microsoft .NET Framework Programming - Jeffrey Richter
C# In a Nutshell - Peter Drayton
Essential .NET - Don Box
ANSI Common Lisp - Paul Graham

About halfway through the L Sharp development, I happened upon the DotLisp project by Rich Hickey and found the source code to be interesting reading and a useful source of ideas and inspiration.

Several people have contributed by providing encouragement, bug reports, comments and fixes; particular thanks to Patrick Dussud, Hans-Peter Wickern and Alex Peake.
